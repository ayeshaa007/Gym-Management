
Gym Management App
A fully offline Android application built using Java + XML + SQLite to help gym owners manage members, memberships, and payments efficiently.

🚀 Features :

Add / Edit / Delete Members

Membership Expiry Tracking

Revenue Calculation

Active & Expired Member Filter

4-Digit PIN Security Lock

Shake Animation on Wrong PIN

Gradient Dashboard UI

Fully Offline (SQLite Database)
